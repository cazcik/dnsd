# utils

simple, dns research

## development

### prerequisites

- [go@latest](https://go.dev)
- [docker@latest](https://docker.com)
- [air](https://github.com/cosmtrek/air)

### getting started

1. clone the repo

```bash
git clone git@github.com:cazcik/utils.git
```

2. start dev env

```bash
air
```
